# Nixfiles

This repo contains the Nix configurations for my PC and MacBook Pro. It was
set up following the dendritic pattern. The following are good references for
learning about the dendritic pattern:

- [Vimjoyer](https://youtu.be/-TRbzkw6Hjs?si=siag0txOXPpXdlM6)
    - Youtube video that got my initial setup off the ground
- [mightyiam](https://github.com/mightyiam/infra)
    - Repo I tried to model my current repo after
- [vic](https://dendrix.denful.dev/Dendritic.html)
    - Small post talking about the dendritic pattern and how it works
